O repositório tem os seguintes programas:

## Trigonometria 
### Simplificar Ângulos Orientados (`angulos_orient.py`)
Reescreve um ângulo orientado (em graus) na forma `x = a + k*360`.

### Converter entre Graus e Radianos (`graus_rad_convert.py`)
Converte um ângulo em graus para um em radianos, e vice-versa.

### Valor Exato das Razões Trigonométricas (`trig_valor_exato.py`)
Escreve o valor exato do resultado das funções trigonométricas.

## Vetores
### Produto Escalar de Vetores com Duas Dimensões (`prod_escalar_2d.py`)
Calcula o produto escalar de dois vetores no plano.

### Produto Escalar de Outros Vetores
Calcula o produto escalar de dois vetores quaisquer (desde que tenham o mesmo número de dimensões)
